# PROYECTO LÓGICA: Katas de Python

Este proyecto forma parte del módulo '6. Python' sobre lógica de programación para el **Máster en Data & Analytics** en **PowerMBA**, donde el objetivo principal es perfeccionar las habilidades de resolución de problemas y afianzar los conceptos fundamentales del lenguaje Python a través de desafíos prácticos.

## ¿Qué es una Kata?

En programación, una **kata** es un ejercicio práctico diseñado para mejorar las habilidades de un programador mediante la repetición y el perfeccionamiento de técnicas de codificación.

---

## Objetivos del Proyecto

El propósito de este repositorio es demostrar el dominio de los pilares de Python:

* **Fundamentos:** Tipos de datos básicos y funciones  *built-in* .
* **Estructuras de Datos:** Listas, diccionarios, tuplas y sets (y sus métodos).
* **Control de Flujo:** Condicionales (`if/else`) y bucles de iteración (`for/while`).
* **Modularidad:** Creación y uso de funciones y módulos.
* **POO:** Programación Orientada a Objetos (Clases y objetos).
* **Calidad:** Aplicación de buenas prácticas y código autodocumentado.

---

## Estructura del Repositorio

El proyecto está organizado de la siguiente manera para facilitar su revisión y ejecución:

* `README.md`: Este archivo, con la descripción y pasos del proyecto.
* `scripts\katas_proyecto.py`: Archivo script que contiene todas las katas resueltas, cada una encabezada por su enunciado correspondiente.
* `notebooks\katas_proyecto.ipynb`: Notebook interactivo con explicaciones detalladas en Markdown y celdas de código independientes para cada ejercicio.

---

## Notas Importantes

**Entendimiento del Código:** Todos los ejercicios incluyen comentarios que detallan la lógica detrás de los pasos más complejos o los desafíos técnicos encontrados.

**Uso de IA:** Se ha hecho un uso moderado y responsable de herramientas de IA para la resolución de dudas puntuales, asegurando en todo momento la comprensión total de la solución implementada y el afianzamiento de los conocimientos.
